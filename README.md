
# TODO APP

A website to write down your to-do list
## Demo


**Webiste:** https://tdlist.vercel.app


## Features

- Create a to-do list
- Persist your to-do list
## Run Locally

Clone the project

```bash
  git clone https://github.com/Mathesu-veLi/todo-app.git
```

Go to the project directory

```bash
  cd todo-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
